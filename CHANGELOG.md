## 0.2.0

- Reimplemented the DocuPass flow in Flutter/Dart instead of hosting native
  DocuPass SDK screens.
- Added Dart API client and controller logic matching the Android flow:
  `get_action`, phone verification, custom form, document selection/upload,
  face upload, contract submit, pending-party refresh, completed, and failed
  routing.
- Added Flutter UI for country selection, document type selection, document
  camera capture, face liveness, phone OTP, custom forms, contract signing,
  pending, success, and failed screens.
- Added an installable Flutter example app with server reference and local
  camera demo flows.
